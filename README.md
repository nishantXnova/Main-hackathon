## 🌟 Key Features

### 🎥 Immersive Experience
- **Dynamic Video Background**: A stunning, full-screen video background that brings the beauty of Nepal directly to your screen.
- **Adaptive "We Need Nepal" Hero Text**: The centerpiece text dynamically analyzes the video background in real-time, adjusting its color (light/dark) to ensure perfect visibility and contrast at all times.
- **Parallax Effects**: Subtle mouse-driven parallax movements create a sense of depth and immersion.
- **Ambient Audio**: Optional ambient soundscapes to enhance the browsing experience.

### 🗺️ interactive Exploration
- **Province Guide**: Detailed information about Nepal's 7 provinces, showcasing their unique landscapes and cultures.
- **Top Places**: A curated list of must-visit destinations like Kathmandu Valley, Pokhara, and Boudhanath Stupa.
- **Place Explorer**: Discover local Mandirs, Heritage sites, Malls, and Marts near you (simulated 3km radius).

### 🛠️ Utilities & Tools
- **Smart Weather**: Real-time weather updates for key locations (Kathmandu, Pokhara, Lukla) to help plan your trips safely.
- **Hospital Locator**: A critical safety feature helping tourists find nearby emergency medical help.
- **Nepal Clock (NPT)**: Always know the local time in Nepal with our custom analog and digital clock widget.
- **AI Chatbot**: An intelligent assistant ready to answer your travel queries instantly.

## 💻 Getting Started

Follow these instructions to get the project up and running on your local machine.

### Prerequisites

- **Node.js** (v14 or higher recommended)
- **npm** (comes with Node.js)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/nishantXnova/Main-hackathon.git
    ```

2.  **Navigate to the project directory**:
    ```bash
    cd "Main hackathon"
    ```

3.  **Install dependencies**:
    ```bash
    npm install
    ```

### Running Locally

To start the development server:

```bash
npm start
```

The application will be available at [http://localhost:3000](http://localhost:3000).

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The build assets will be located in the `build/` directory.
